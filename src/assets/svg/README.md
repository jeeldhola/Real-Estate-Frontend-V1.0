# SVG Assets Folder

Place all your SVG assets in this folder:
- `logo.svg` (For the main website and sign-in logo)
- `background.svg` (For custom backgrounds)
- `banner.svg` (For promo or dashboard banners)
- Other SVG elements

You can reference these assets in your code using:
`import logoUrl from "@/assets/svg/logo.svg";` or relative paths.
